# CHORSU — Иммерсивная презентация

> «Мы идём на базар вместо вас.»
> Цифровая инфраструктура для традиционных базаров. Нукус, Узбекистан.

Кинематографичный скролл-сайт в стиле keynote (Apple / Linear / Stripe): 13 глав, плавный
скролл Lenis, анимации GSAP ScrollTrigger, editorial-типографика Fraunces + Inter,
палитра «глина и специи».

---

## 🚀 Запуск

Никакой сборки не нужно. Достаточно открыть `index.html`.

Для корректной работы шрифтов, видео и `<img>` лучше поднять локальный сервер:

```bash
cd project
python3 -m http.server 8000
# затем открыть http://localhost:8000
```

Библиотеки (GSAP, ScrollTrigger, Lenis, SplitType, Lucide, Tailwind) подключены по CDN —
нужен интернет при первом открытии.

| Действие | Как |

с подписью. Как только вы положите файл с нужным именем, плейсхолдер автоматически скроется.

## ✏️ Что и где менять

- **Текст глав** — прямо в `index.html` (всё на русском, семантическая разметка).
- **Цифры и метрики** (рынок, бизнес-модель, инвестиции) — атрибуты `data-count`,
  `data-prefix`, `data-suffix` у чисел в `index.html`. Строки, помеченные
  `* Иллюстративные…`, — это **плейсхолдеры**, замените на реальные данные.
- **Палитра и типографика** — переменные `:root` в начале `style.css`
  (`--primary`, `--accent`, `--bg`, тёмная тема `[data-theme="dark"]` и т.д.).
- **Контакты и QR** — глава 13 в `index.html` (`chorsu.uz`, `@chorsu`, почта).
  QR — стилизованный плейсхолдер `.ph--qr`; замените на `<img>` с реальным кодом.

## ♿ Доступность и производительность

- Полная поддержка `prefers-reduced-motion` — при включённом «уменьшить движение»
  анимации отключаются, контент показывается сразу.
- Анимации только на `transform` / `opacity` (GPU), цель — 60 FPS.
- Ленивая загрузка изображений (`loading="lazy"`), семантические теги, фокус-стили.
- Адаптив: десктоп → планшет → мобильный (горизонтальный скролл продукта на мобильном
  превращается в вертикальный список).

## 🎨 Дизайн-концепция

**«Базар, не спеша»** — тепло редакционного журнала о еде и точность keynote-презентации.
Базар здесь — не «проблема для дизрапта», а живая традиция, которой служит продукт.
Глиняные фоны, солнечный свет, продукты, снятые как для гастрономического журнала;
ритм светлых и тёмных глав держит кинематографичность.

- **Шрифты:** Fraunces (serif, заголовки) + Inter (гуманистический sans, текст).
- **Цвета:** терракота/специи, травяная зелень, шафран, инжир — без «стартап-зелёного».
- **Движение:** маскированные ревилы текста, клип-ревилы изображений, счётчики,
  рисование маршрута доставки, параллакс, микровзаимодействия.
